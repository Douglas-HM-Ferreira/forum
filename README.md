# Forum API

Fórum moderno de suporte às APIs bancárias (PIX, Boleto, CNAB, Banking, OpenFinance), com automação de atendimento por IA.

## Stack

- Java 21
- Spring Boot 3.2
- Spring Security + JWT
- PostgreSQL 16
- Flyway
- Docker
- Swagger / OpenAPI

## Roadmap

- [x] Fase 1 — Fundação (Spring Boot + PostgreSQL + JWT)
- [ ] Fase 2 — Busca e qualidade (Elasticsearch + Redis)
- [ ] Fase 3 — IA no atendimento (RAG + LLM)
- [ ] Fase 4 — IA avançada (Fine-tuning + Agentes)
